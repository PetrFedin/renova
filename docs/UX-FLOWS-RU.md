# UX Renova iPhone

## Принципы
- Крупные цифры (бюджет, %, дни)
- Один primary action на экран
- Шаблоны вместо пустых форм
- Зелёный/жёлтый/красный для контроля

## Заказчик: Главная
- Progress bar %, бюджет факт/план
- Карточка «Следующий шаг» (приёмка)
- Алерты перерасхода/просрочки

## Исполнитель: Объекты
- Список проектов со статусами
- Быстрое создание сметы из шаблона
- Таб «Деньги»: чеки, Мой налог
