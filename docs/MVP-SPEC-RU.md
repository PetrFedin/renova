# Renova MVP — спецификация v1.0

> Цель: iPhone-приложение, где **всё крутится вокруг расчётов и контроля** — смета, факт, сроки, расходники, чеки, рейтинг.

## 1. Принципы

1. Один экран — один ответ: «Сколько потрачено?», «На каком этапе?», «Что дальше?»
2. Каждое действие обновляет цифры
3. Заказчик бесплатно, исполнитель — подписка
4. Легальность = рейтинг + доверие

## 2. Модули MVP

### M1. Проект и помещения
- Шаблоны: косметический / капитальный / ванная / кухня
- Комнаты: Д×Ш×В → авто S_пол, S_стен, периметр
- Запас: +10% плитка, +15% обои, +5% краска

### M2. Смета (ядро)
- Строки: материал | работа
- Нормы из справочника (30 работ MVP)
- Версии сметы, план vs факт, PDF

### M3. Этапы
- Kanban: запланирован → в работе → приёмка → сдан
- Фото, чек-лист, алерт просрочки

### M4. Финансы
- Платежи по этапам, чеки (QR), учёт расходников
- Перерасход >10% → алерт

### M5. ФНС
- L1: проверка ИНН самозанятого (MVP)
- L2: проверка чека QR (MVP)
- L3: OAuth «Мой налог» (v1.1)

### M6. Рейтинг
- Сроки 30% + бюджет 25% + качество 25% + чеки 20%

## 3. Ценность заказчику
- Калькулятор за 3 мин | Поэтапная оплата | Audit trail | Контроль расходников

## 4. Ценность исполнителю
- Смета из шаблона | Авто-чеки НПД | Лиды | CRM объектов | Рейтинг

## 5. iPhone — 4 таба

**Заказчик:** Главная | Смета | Этапы | Профиль
**Исполнитель:** Объекты | Смета | Деньги | Профиль
