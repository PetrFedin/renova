# P3-W35 — Sprint 3 Chains (сквозные цепочки)

**Цель:** закрыть обрывы golden path без новых экранов.

## Сделано

| Цепочка | Было | Стало |
|---------|------|--------|
| CO → approve | Notify на budget (мимо сметы) | Notify → смета `estimateLayer=changes` |
| CO → budget + eSign | Budget line + draft без UI CTA | Alert: бюджет + **Подписать**; `document_id` в API; notify на `/documents` |
| Portal | Слабый бренд; accept без pay hint | Hero RENOVA + прогресс + todo; после приёмки — hint на оплату |
| Punch photo | Только камера | Камера → галерея fallback; OfflineSync на плане |
| Docs offline | Sign без offline UX | `withBusy` → `notifyOfflineQueued` |
| Stage → calendar | Нет `ical_uid` / started event | `ical_uid` на start; event `stage_started` |

## DoD
- CO create → customer opens estimate changes
- CO approve → budget delta + sign path
- Portal accept → pay messaging
- Punch works without camera permission (gallery)
- Stage start visible on calendar hub

## Scope
Только Renova. Syntha вне скоупа.
