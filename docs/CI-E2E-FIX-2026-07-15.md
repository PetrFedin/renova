# CI e2e fix — 2026-07-15

## Симптом

GitHub Actions job `e2e` → **FAILURE**:

```text
uvicorn: command not found
```

Poetry ставил зависимости во venv, а CI запускал `uvicorn` из системного PATH.

## Исправление

`.github/workflows/ci.yml`:

- `poetry run uvicorn …`
- ожидание `/health` перед `e2e-smoke.sh`
- отдельный шаг `poetry run alembic upgrade head` на service Postgres (staging-like)

Локальный `bash scripts/e2e-smoke.sh` и `npm run merge:check` оставались green.
